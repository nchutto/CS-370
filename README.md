# CS 370: Pirate Intelligent Agent – Reinforcement Learning Project

## Project Overview

This project showcases the development of a Pirate Intelligent Agent using reinforcement learning and neural networks.  
It demonstrates how an agent can learn to navigate a maze and reach a goal through experience and iterative improvement.

The project focuses on:

- Reinforcement learning concepts  
- Neural network-based decision making  
- Trial-and-error learning through environment interaction  
- Performance improvement over multiple training epochs  

---

## Key Concepts Applied

- **Q-learning**
- **Experience replay**
- **Exploration vs. exploitation (epsilon tuning)**
- **Model training and updates**
- **State-action decision making**

---

## Work Completed

The project involved working from a provided Jupyter Notebook that included:

- Environment setup

- Applying Q-learning and experience replay
